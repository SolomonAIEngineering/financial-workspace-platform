# PricingSection Component

The `PricingSection` component displays pricing plans and options on landing pages. It typically presents multiple tiers in a comparative format, highlighting features, limitations, and price points for each plan. The section often includes toggles for monthly/annual billing and call-to-action buttons for each plan.

## Usage Example

Here's a basic usage example of the `PricingSection`:

```tsx
<Canvas>
  <Story name="Default">
    <PricingSection
      title="Simple, Transparent Pricing"
      subtitle="Choose the plan that's right for your business"
      plans={[
        {
          name: "Starter",
          description: "Perfect for small businesses just getting started",
          price: {
            monthly: 29,
            annual: 290
          },
          features: [
            "Basic financial analytics",
            "Up to 100 transactions/month",
            "Email support",
            "1 user account"
          ],
          ctaLabel: "Start with Starter",
          ctaHref: "/signup?plan=starter",
          highlighted: false
        },
        {
          name: "Professional",
          description: "Ideal for growing businesses with more needs",
          price: {
            monthly: 79,
            annual: 790
          },
          features: [
            "Advanced financial analytics",
            "Unlimited transactions",
            "Priority support",
            "5 user accounts",
            "Custom reporting"
          ],
          ctaLabel: "Choose Professional",
          ctaHref: "/signup?plan=professional",
          highlighted: true
        },
        {
          name: "Enterprise",
          description: "For large businesses with complex requirements",
          price: {
            monthly: 199,
            annual: 1990
          },
          features: [
            "Complete financial suite",
            "Unlimited everything",
            "24/7 dedicated support",
            "Unlimited user accounts",
            "Custom integrations",
            "Dedicated account manager"
          ],
          ctaLabel: "Contact Sales",
          ctaHref: "/contact-sales",
          highlighted: false
        }
      ]}
      currency="USD"
      defaultBillingPeriod="monthly"
    />
  </Story>
</Canvas>
```
