# DataTablePagination Component

The `DataTablePagination` component provides controls for navigating through pages of data in a table. It includes page selection buttons, a page size selector, and information about the current range of records being displayed. The component helps users efficiently browse large datasets without overwhelming the interface.

## Usage Example

Here's a basic usage example of the `DataTablePagination`:

```tsx
<Canvas>
  <Story name="Default">
    <DataTablePagination
      table={{
        getState: () => ({
          pagination: {
            pageIndex: 0,
            pageSize: 10
          }
        }),
        getPageCount: () => 5,
        getRowModel: () => ({
          rows: Array(10).fill(null)
        }),
        getFilteredRowModel: () => ({
          rows: Array(50).fill(null)
        }),
        setPageIndex: (index) => console.info("Go to page:", index),
        setPageSize: (size) => console.info("Set page size:", size),
        previousPage: () => console.info("Previous page"),
        nextPage: () => console.info("Next page")
      }}
      pageSizeOptions={[5, 10, 20, 50, 100]}
      showRowsInfo={true}
      className="p-2 border-t"
    />
  </Story>
</Canvas>
```
