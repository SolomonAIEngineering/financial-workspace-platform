# UserNav Component

The `UserNav` component provides a user navigation interface typically shown in the header of the transaction table area. It displays the current user's information, such as avatar and name, and includes a dropdown menu with user-related actions like profile settings, account management, and sign out functionality.

## Usage Example

Here's a basic usage example of the `UserNav`:

```tsx
<Canvas>
  <Story name="Default">
    <UserNav
      user={{
        name: "John Doe",
        email: "john.doe@example.com",
        avatar: "/avatars/john-doe.jpg"
      }}
      items={[
        { 
          label: "Profile", 
          icon: "UserIcon", 
          onClick: () => console.info("Navigate to profile") 
        },
        { 
          label: "Settings", 
          icon: "SettingsIcon", 
          onClick: () => console.info("Open settings") 
        },
        { 
          label: "Sign out", 
          icon: "LogOutIcon", 
          onClick: () => console.info("Sign out"),
          separator: true
        }
      ]}
      className="ml-auto"
    />
  </Story>
</Canvas>
```
