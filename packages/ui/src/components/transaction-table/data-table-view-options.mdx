# DataTableViewOptions Component

The `DataTableViewOptions` component provides controls for customizing the display of a data table. It allows users to toggle the visibility of columns, rearrange columns, and save view preferences. This component helps users tailor the table view to focus on the data most relevant to their current task.

## Usage Example

Here's a basic usage example of the `DataTableViewOptions`:

```tsx
<Canvas>
  <Canvas of={DataTableViewOptions.Default}>
    <DataTableViewOptions
      table={{
        getAllColumns: () => [
          { 
            id: "date", 
            getCanHide: () => true, 
            getIsVisible: () => true, 
            toggleVisibility: () => console.info("Toggle date column")
          },
          { 
            id: "description", 
            getCanHide: () => true, 
            getIsVisible: () => true, 
            toggleVisibility: () => console.info("Toggle description column")
          },
          { 
            id: "amount", 
            getCanHide: () => true, 
            getIsVisible: () => true, 
            toggleVisibility: () => console.info("Toggle amount column")
          },
          { 
            id: "category", 
            getCanHide: () => true, 
            getIsVisible: () => false, 
            toggleVisibility: () => console.info("Toggle category column")
          }
        ]
      }}
      saveViewHandler={(view) => console.info("Save view:", view)}
      className="w-60"
    />
  </Canvas>
</Canvas>
```
