# DataTableFilters Component

The `DataTableFilters` component provides a user interface for filtering data in a table. It offers various filter types like text, date range, numeric range, and select filters. The component makes it easy for users to narrow down large datasets and find specific information quickly and efficiently.

## Usage Example

Here's a basic usage example of the `DataTableFilters`:

```tsx
<Canvas>
  <Canvas of={DataTableFilters.Default}>
    <DataTableFilters
      table={{
        getColumn: (id) => ({
          id,
          getFilterValue: () => undefined,
          setFilterValue: (value) => console.info(`Set filter ${id}:`, value)
        }),
        getAllColumns: () => [
          { id: "date", getFacetedUniqueValues: () => new Map() },
          { id: "amount", getFacetedUniqueValues: () => new Map() },
          { id: "category", getFacetedUniqueValues: () => new Map() },
          { id: "status", getFacetedUniqueValues: () => new Map() }
        ]
      }}
      filters={[
        { id: "date", type: "date" },
        { id: "amount", type: "number" },
        { id: "category", type: "select" },
        { id: "status", type: "select" }
      ]}
      className="p-4 border rounded"
    />
  </Canvas>
</Canvas>
```
