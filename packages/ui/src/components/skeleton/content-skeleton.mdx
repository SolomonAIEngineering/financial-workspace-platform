# ContentSkeleton Component

The `ContentSkeleton` component provides a loading placeholder that mimics the structure of content while it's being loaded. It displays animated placeholder elements that match the layout of the expected content, improving perceived performance and reducing layout shifts when content becomes available.

## Usage Example

Here's a basic usage example of the `ContentSkeleton`:

```tsx
<Canvas>
  <Canvas of={ContentSkeleton.Default}>
    <ContentSkeleton 
      headerVisible={true}
      paragraphCount={3}
      imageVisible={true}
      cardLayout={true}
      className="max-w-xl"
    />
  </Canvas>
</Canvas>
```
