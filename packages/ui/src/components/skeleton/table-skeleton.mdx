# TableSkeleton Component

The `TableSkeleton` component provides a loading placeholder for a data table. It displays animated skeleton rows and columns that mimic the structure of the table that will eventually be loaded. This component helps maintain visual consistency during data loading and reduces layout shifts when the actual table data becomes available.

## Usage Example

Here's a basic usage example of the `TableSkeleton`:

```tsx

  <Story name="Default">
``` 
