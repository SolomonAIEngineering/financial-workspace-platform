# DashboardSkeleton Component

The `DashboardSkeleton` component provides a loading placeholder for an entire dashboard layout. It displays animated skeleton elements that represent various dashboard components like charts, tables, metrics, and navigation elements. This component helps maintain a consistent layout during data loading and improves perceived performance.

## Usage Example

Here's a basic usage example of the `DashboardSkeleton`:

```tsx
<Canvas>
  <Canvas of={DashboardSkeleton.Default}>
    <DashboardSkeleton 
      sidebarVisible={true}
      chartCount={2}
      tableVisible={true}
      metricCount={4}
      className="w-full h-screen"
    />
  </Canvas>
</Canvas>
```
