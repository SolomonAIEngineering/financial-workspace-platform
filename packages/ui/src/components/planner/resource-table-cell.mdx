# ResourceTableCell Component

The `ResourceTableCell` component displays resource information in the planner's resource table column. It shows the resource name, optional icon or image, and additional metadata, creating a visual index for the resources displayed in the timeline. The component can be styled differently based on resource status or availability.

## Usage Example

Here's a basic usage example of the `ResourceTableCell`:

```tsx
<Canvas>
  <Canvas of={ResourceTableCell.Default}>
    <ResourceTableCell
      resource={{
        id: "1",
        title: "Conference Room A",
        capacity: 12,
        available: true
      }}
      onResourceClick={(resourceId) => console.info("Resource clicked:", resourceId)}
      className="border p-2"
    />
  </Canvas>
</Canvas>
```
