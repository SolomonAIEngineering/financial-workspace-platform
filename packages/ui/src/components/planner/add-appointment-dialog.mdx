# AddAppointmentDialog Component

The `AddAppointmentDialog` component provides a modal interface for creating or editing appointments in the planner. It includes form fields for title, description, date/time selection, resource assignment, and other appointment details, with validation to ensure all required information is provided.

## Usage Example

Here's a basic usage example of the `AddAppointmentDialog`:

```tsx
<Canvas>
  <Canvas of={AddAppointmentDialog.Default}>
    <AddAppointmentDialog
      open={true}
      onClose={() => console.info("Dialog closed")}
      onSave={(appointmentData) => {
        console.info("Appointment saved:", appointmentData);
        return Promise.resolve();
      }}
      initialData={{
        title: "",
        startDate: new Date(),
        endDate: new Date(Date.now() + 60 * 60 * 1000),
        resourceId: ""
      }}
      resources={[
        { id: "1", title: "Conference Room A" },
        { id: "2", title: "Conference Room B" },
        { id: "3", title: "Meeting Room 101" }
      ]}
      isNewAppointment={true}
    />
  </Canvas>
</Canvas>
```
