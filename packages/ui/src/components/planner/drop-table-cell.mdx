# DropTableCell Component

The `DropTableCell` component represents a droppable cell in the planner grid where appointments can be created or moved. It handles drag-and-drop interactions, providing visual feedback when an appointment is being dragged over it, and triggers appropriate events when an appointment is dropped onto the cell.

## Usage Example

Here's a basic usage example of the `DropTableCell`:

```tsx
<Canvas>
  <Story name="Default">
    <DropTableCell
      resourceId="resource-1"
      startTime={new Date(2023, 5, 15, 10, 0)}
      endTime={new Date(2023, 5, 15, 11, 0)}
      onAppointmentDrop={(data) => console.info("Appointment dropped:", data)}
      onDoubleClick={() => console.info("Cell double-clicked")}
      className="border border-gray-200 p-2"
    />
  </Story>
</Canvas>
```
