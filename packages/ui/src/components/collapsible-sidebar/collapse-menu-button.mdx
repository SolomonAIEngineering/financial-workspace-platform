# CollapseMenuButton Component

The `CollapseMenuButton` is a toggleable button component designed for collapsible navigation menus. It provides a user-friendly way to expand or collapse sidebar menus, helping to optimize screen space while maintaining easy access to navigation options.

## Usage Example

Here's a basic usage example of the `CollapseMenuButton`:

```tsx
<Canvas>
  <Canvas of={CollapseMenuButton.Default}>
    <CollapseMenuButton
      collapsed={false}
      onClick={() => console.info("Toggle menu collapse")}
      className="my-2"
    />
  </Canvas>
</Canvas>
```
