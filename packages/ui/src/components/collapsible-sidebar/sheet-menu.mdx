# SheetMenu Component

The `SheetMenu` component provides a mobile-friendly slide-out navigation menu. It's typically used as an alternative to the sidebar on smaller screens, presenting the same navigation options in a format optimized for touch interaction and limited screen space.

## Usage Example

Here's a basic usage example of the `SheetMenu`:

```tsx
<Canvas>
  <Story name="Default">
    <SheetMenu
      open={true}
      onOpenChange={(open) => console.log("Sheet menu open:", open)}
      items={[
        { title: "Dashboard", icon: "HomeIcon", href: "/dashboard" },
        { title: "Analytics", icon: "BarChartIcon", href: "/analytics" },
        { title: "Settings", icon: "SettingsIcon", href: "/settings" }
      ]}
      onMenuItemClick={(item) => console.log("Clicked:", item.title)}
    />
  </Story>
</Canvas>
```
