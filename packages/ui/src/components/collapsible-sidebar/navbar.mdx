# Navbar Component

The `Navbar` component provides a horizontal navigation bar typically displayed at the top of the application. It works in conjunction with the collapsible sidebar to provide comprehensive navigation options, and often includes branding, search functionality, and user-related controls.

## Usage Example

Here's a basic usage example of the `Navbar`:

```tsx
<Canvas>
  <Canvas of={Navbar.Default}>
    <Navbar
      logoSrc="/logo.svg"
      title="Application"
      onMenuClick={() => console.info("Menu clicked")}
    >
      <div className="flex items-center space-x-4">
        <button className="p-2">Search</button>
        <button className="p-2">Notifications</button>
        <button className="p-2">Profile</button>
      </div>
    </Navbar>
  </Canvas>
</Canvas>
```
