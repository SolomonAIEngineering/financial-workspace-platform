# UserNav Component

The `UserNav` component provides user-related navigation and actions, typically displayed in the navbar or sidebar. It includes a user avatar, name, and a dropdown menu with options like profile settings, account management, and sign out functionality.

## Component Props

<Canvas of={UserNav.Default} />

<Controls />

## Usage Example

Here's a basic usage example of the `UserNav`:

```tsx
<Canvas>
  <Canvas of={UserNav.Default}>
    <UserNav
      user={{
        name: "John Doe",
        email: "john.doe@example.com",
        avatar: "/avatars/john-doe.jpg"
      }}
      onSignOut={() => console.info("Sign out")}
      onProfileClick={() => console.info("View profile")}
      onSettingsClick={() => console.info("Open settings")}
    />
  </Canvas>
</Canvas>
```
