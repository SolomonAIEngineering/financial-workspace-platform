# CollapsiblePanelLayout Component

The `CollapsiblePanelLayout` component provides a complete layout framework combining a collapsible sidebar with a main content area. It manages the state and interaction between the sidebar and content areas, ensuring they work together seamlessly as the sidebar expands and collapses.

## Component Props

<Canvas of={CollapsiblePanelLayout.Default} />

<Controls />

## Usage Example

Here's a basic usage example of the `CollapsiblePanelLayout`:

```tsx
<Canvas>
  <Canvas of={CollapsiblePanelLayout.Default}>
    <CollapsiblePanelLayout
      sidebarContent={
        <div className="p-4">
          <h2 className="text-lg font-semibold mb-4">Navigation</h2>
          <ul className="space-y-2">
            <li>Dashboard</li>
            <li>Analytics</li>
            <li>Settings</li>
          </ul>
        </div>
      }
      mainContent={
        <div className="p-6">
          <h1 className="text-2xl font-bold mb-4">Main Content</h1>
          <p>This is the main content area of the application.</p>
        </div>
      }
      defaultCollapsed={false}
    />
  </Canvas>
</Canvas>
```
