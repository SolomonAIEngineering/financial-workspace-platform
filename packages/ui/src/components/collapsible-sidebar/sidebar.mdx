# Sidebar Component

The `Sidebar` component provides a collapsible navigation panel for web applications. It serves as a container for navigation menus, logos, and other sidebar elements, and can be toggled between expanded and collapsed states to optimize screen space while maintaining navigation functionality.

## Usage Example

Here's a basic usage example of the `Sidebar`:

```tsx
<Canvas>
  <Story name="Default">
    <Sidebar 
      collapsed={false}
      onCollapse={() => console.log("Sidebar collapsed")}
      className="h-screen"
    >
      <div className="p-4 border-b">
        <h2 className="text-lg font-semibold">Application</h2>
      </div>
      {/* Navigation menu would go here */}
      <div className="p-4">
        <p>Sidebar content</p>
      </div>
    </Sidebar>
  </Story>
</Canvas>
```
