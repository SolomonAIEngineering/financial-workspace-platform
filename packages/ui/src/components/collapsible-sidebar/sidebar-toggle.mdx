# SidebarToggle Component

The `SidebarToggle` component provides a button for toggling the expanded/collapsed state of a sidebar. It typically displays different icons based on the current state of the sidebar, making it clear to users what action will be performed when clicked.

## Usage Example

Here's a basic usage example of the `SidebarToggle`:

```tsx
<Canvas>
  <Story name="Default">
    <SidebarToggle
      collapsed={false}
      onClick={() => console.log("Toggle sidebar")}
      className="mb-4"
    />
  </Story>
</Canvas>
```
