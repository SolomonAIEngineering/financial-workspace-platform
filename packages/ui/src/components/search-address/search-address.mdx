# SearchAddress Component

The `SearchAddress` component provides an interactive address search and selection interface. It allows users to input and search for addresses with autocomplete functionality, select from suggestions, and provides validation for complete addresses. This component is typically used in forms where address input is required.

## Usage Example

Here's a basic usage example of the `SearchAddress`:

```tsx
<Canvas>
  <Canvas of={SearchAddress.Default}>
    <SearchAddress
      apiKey="your_google_places_api_key"
      onAddressSelect={(address) => console.info("Selected address:", address)}
      placeholder="Enter your address"
      label="Address"
      required={true}
    />
  </Canvas>
</Canvas>
```
