# CreditAccountCard Component

The `CreditAccountCard` component provides a comprehensive card display for credit account information. It shows the current balance, available credit, credit limit, payment due date, and recent transactions. This component is designed to give users a quick overview of their credit card or line of credit status, utilization, and recent activity.

## Usage Example

Here's a basic usage example of the `CreditAccountCard`:

```tsx
<Canvas>
  <Canvas of={CreditAccountCard.Default}>
    <CreditAccountCard
      account={{
        id: "crd-456",
        name: "Business Visa",
        type: "CREDIT_CARD",
        subtype: "BUSINESS",
        balance: -3245.75,
        availableCredit: 6754.25,
        creditLimit: 10000.00,
        currency: "USD",
        lastFourDigits: "1234",
        dueDate: "2023-06-15",
        minimumPayment: 125.00,
        apr: 16.99,
        institution: {
          name: "Capital One",
          logo: "/logos/capital-one.png"
        }
      }}
      transactions={[
        {
          id: "tx-1",
          date: "2023-05-15",
          description: "Office Supplies",
          amount: -245.50,
          category: "Office Expenses"
        },
        {
          id: "tx-2",
          date: "2023-05-14",
          description: "Business Lunch",
          amount: -68.25,
          category: "Meals"
        },
        {
          id: "tx-3",
          date: "2023-05-13",
          description: "Software License",
          amount: -89.99,
          category: "Software"
        }
      ]}
      onViewDetails={() => console.info("View account details")}
      onMakePayment={() => console.info("Make payment")}
      onViewStatement={() => console.info("View statement")}
      onViewAllTransactions={() => console.info("View all transactions")}
      loading={false}
    />
  </Canvas>
</Canvas>
``` 
