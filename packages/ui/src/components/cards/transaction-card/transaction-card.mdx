# TransactionCard Component

The `TransactionCard` component provides a card interface for displaying detailed information about a financial transaction. It shows transaction date, amount, description, category, and status. The component can be used in transaction lists, details pages, or for individual transaction review.

<Canvas of={TransactionCard.Default} />

<Controls />

## Usage Example

Here's a basic usage example of the `TransactionCard`:

```tsx
<Canvas>
  <Canvas of={TransactionCard.Default}>
    <TransactionCard
      transaction={{
        id: "tx-123",
        date: "2023-05-15",
        description: "Office Supplies - Staples",
        amount: -245.50,
        currency: "USD",
        category: "Office Expenses",
        subcategory: "Office Supplies",
        status: "completed",
        merchantName: "Staples",
        merchantLogo: "/logos/staples.png",
        paymentMethod: "Credit Card (...1234)",
        notes: "Purchased office supplies for new employees"
      }}
      onViewDetails={() => console.info("View transaction details")}
      onEditCategory={() => console.info("Edit transaction category")}
      onAddNote={() => console.info("Add transaction note")}
      onFlagReview={() => console.info("Flag for review")}
      loading={false}
    />
  </Canvas>
</Canvas>
```
