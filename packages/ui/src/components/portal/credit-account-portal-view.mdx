# CreditAccountPortalView Component

The `CreditAccountPortalView` component provides a detailed view for a specific credit account or credit card. It displays account information, balance history, transaction history, payment due dates, and utilization metrics. This component enables users to monitor and manage their credit accounts effectively.

## Usage Example

Here's a basic usage example of the `CreditAccountPortalView`:

```tsx
<Canvas>
  <Story name="Default">
    <CreditAccountPortalView
      account={{
        id: "crd-456",
        name: "Business Visa",
        type: "CREDIT_CARD",
        balance: -3245.75,
        availableCredit: 6754.25,
        creditLimit: 10000.00,
        currency: "USD",
        lastFourDigits: "1234",
        dueDate: "2023-06-15",
        minimumPayment: 125.00,
        apr: 16.99,
        institution: {
          name: "Capital One",
          logo: "/logos/capital-one.png"
        }
      }}
      transactions={[
        {
          id: "tx-1",
          date: "2023-05-15",
          description: "Office Supplies",
          amount: -245.50,
          category: "Office Expenses",
          pending: false
        },
        {
          id: "tx-2",
          date: "2023-05-14",
          description: "Business Lunch",
          amount: -68.25,
          category: "Meals",
          pending: false
        },
        {
          id: "tx-3",
          date: "2023-05-16",
          description: "Software License",
          amount: -89.99,
          category: "Software",
          pending: true
        }
      ]}
      balanceHistory={[
        { date: "2023-05-01", balance: -2500.00 },
        { date: "2023-05-05", balance: -2750.25 },
        { date: "2023-05-10", balance: -3100.50 },
        { date: "2023-05-15", balance: -3245.75 }
      ]}
      onTransactionSelect={(txId) => console.info("Transaction selected:", txId)}
      onMakePayment={() => console.info("Make payment")}
      onViewStatement={() => console.info("View statement")}
      loading={false}
    />
  </Story>
</Canvas>
``` 
