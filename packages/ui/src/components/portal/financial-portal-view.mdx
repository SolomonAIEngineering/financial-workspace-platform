# FinancialPortalView Component

The `FinancialPortalView` component provides a comprehensive dashboard for viewing and managing financial information. It aggregates data from various financial accounts and displays them in a unified interface with summaries, charts, and actionable insights. This component serves as the main entry point for users to monitor their overall financial health.

## Usage Example

Here's a basic usage example of the `FinancialPortalView`:

```tsx
<Canvas>
  <Canvas of={FinancialPortalView.Default}>
    <FinancialPortalView
      accounts={[
        {
          id: "acc-123",
          name: "Main Checking",
          type: "CHECKING",
          balance: 12450.75,
          currency: "USD",
          institution: {
            name: "Chase Bank",
            logo: "/logos/chase.png"
          }
        },
        {
          id: "acc-456",
          name: "Business Savings",
          type: "SAVINGS",
          balance: 45250.00,
          currency: "USD",
          institution: {
            name: "Bank of America",
            logo: "/logos/bofa.png"
          }
        }
      ]}
      transactions={[
        {
          id: "tx-1",
          date: "2023-05-15",
          description: "Office Supplies",
          amount: -245.50,
          category: "Office Expenses"
        },
        {
          id: "tx-2",
          date: "2023-05-14",
          description: "Client Payment",
          amount: 1200.00,
          category: "Income"
        }
      ]}
      onAccountSelect={(accountId) => console.info("Selected account:", accountId)}
      onViewAllTransactions={() => console.info("View all transactions")}
      onAddAccount={() => console.info("Add new account")}
      loading={false}
    />
  </Canvas>
</Canvas>
```
