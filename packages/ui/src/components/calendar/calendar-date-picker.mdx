# CalendarDatePicker Component

The `CalendarDatePicker` is a flexible and customizable date range picker component. It allows users to select a date range with options for predefined ranges like "Last 7 days", "Last month", etc. The component includes month and year navigation, timezone support, and provides a visually intuitive way to select date ranges.

## Usage Example

Here's a basic usage example of the `CalendarDatePicker`:

```tsx
<Canvas>
  <Story name="Default">
    <CalendarDatePicker 
      date={{
        from: new Date(2023, 0, 1),
        to: new Date()
      }}
      onDateSelect={(range) => console.info("Selected range:", range)}
      numberOfMonths={2}
      yearsRange={5}
    />
  </Story>
</Canvas>
```
