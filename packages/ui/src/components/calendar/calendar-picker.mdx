# CalendarPicker Component

The `CalendarPicker` is a form-based date range picker component that wraps the `CalendarDatePicker` for easy integration with form-based workflows. It provides a clean interface for users to select date ranges and includes form validation using Zod.

## Usage Example

Here's a basic usage example of the `CalendarPicker`:

```tsx
<Canvas>
  <Story name="Default">
    <CalendarPicker 
      initialFrom={new Date(2023, 0, 1)}
      initialTo={new Date()}
      onDateChange={(from, to) => console.log("Date range changed:", from, to)}
    />
  </Story>
</Canvas>
```
