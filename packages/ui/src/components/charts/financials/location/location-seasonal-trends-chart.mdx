# LocationSeasonalTrendsChart Component

The `LocationSeasonalTrendsChart` visualizes how spending patterns at different locations vary by season or time of year. This chart helps businesses identify seasonal trends in location-based spending, allowing for better inventory management, staffing decisions, and marketing strategies based on geographic seasonal patterns.

<Controls />

## Usage Example

Here's a basic usage example of the `LocationSeasonalTrendsChart`:

```tsx
<Canvas>
  <Story name="Default">
    <LocationSeasonalTrendsChart 
      data={[
        { season: "Winter", newYork: 12000, losAngeles: 18000, chicago: 9000 },
        { season: "Spring", newYork: 15000, losAngeles: 16000, chicago: 14000 },
        { season: "Summer", newYork: 18000, losAngeles: 14000, chicago: 19000 },
        { season: "Fall", newYork: 14000, losAngeles: 15000, chicago: 13000 }
      ]}
      locations={["newYork", "losAngeles", "chicago"]}
      displayNames={{
        newYork: "New York",
        losAngeles: "Los Angeles",
        chicago: "Chicago"
      }}
      currency="USD"
      height={400}
    />
  </Story>
</Canvas>
```
