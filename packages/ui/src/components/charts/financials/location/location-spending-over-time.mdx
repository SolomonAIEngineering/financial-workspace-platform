# LocationSpendingOverTime Component

The `LocationSpendingOverTime` chart tracks spending patterns at different geographic locations over time. This visualization helps businesses monitor how their spending evolves across various locations, identify growing or declining markets, and make data-driven decisions about resource allocation and expansion strategies.

## Usage Example

Here's a basic usage example of the `LocationSpendingOverTime`:

```tsx
<Canvas>
  <Story name="Default">
    <LocationSpendingOverTime 
      data={[
        { 
          date: new Date(2023, 0, 1), 
          newYork: 42000, 
          sanFrancisco: 38000, 
          austin: 25000 
        },
        { 
          date: new Date(2023, 1, 1), 
          newYork: 45000, 
          sanFrancisco: 41000, 
          austin: 28000 
        },
        { 
          date: new Date(2023, 2, 1), 
          newYork: 43000, 
          sanFrancisco: 40000, 
          austin: 30000 
        }
      ]}
      locations={["newYork", "sanFrancisco", "austin"]}
      displayNames={{
        newYork: "New York",
        sanFrancisco: "San Francisco",
        austin: "Austin"
      }}
      currency="USD"
      height={400}
    />
  </Story>
</Canvas>
```
