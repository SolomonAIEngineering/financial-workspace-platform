# MerchantSeasonalTrendsChart Component

The `MerchantSeasonalTrendsChart` visualizes how spending with different merchants varies throughout the year or by season. This chart helps businesses identify seasonal spending patterns with their vendors, allowing for better inventory planning, budget forecasting, and negotiation of seasonal terms.

## Component Props

<Canvas of={MerchantSeasonalTrendsChart.Default} />

<Controls />

## Usage Example

Here's a basic usage example of the `MerchantSeasonalTrendsChart`:

```tsx
<Canvas>
  <Canvas of={MerchantSeasonalTrendsChart.Default}>
    <MerchantSeasonalTrendsChart 
      data={[
        { season: "Q1", supplierA: 28000, supplierB: 22000, supplierC: 18000 },
        { season: "Q2", supplierA: 31000, supplierB: 24000, supplierC: 21000 },
        { season: "Q3", supplierA: 25000, supplierB: 28000, supplierC: 24000 },
        { season: "Q4", supplierA: 35000, supplierB: 30000, supplierC: 29000 }
      ]}
      merchants={["supplierA", "supplierB", "supplierC"]}
      displayNames={{
        supplierA: "Supplier A",
        supplierB: "Supplier B",
        supplierC: "Supplier C"
      }}
      currency="USD"
      height={400}
    />
  </Canvas>
</Canvas>
```
