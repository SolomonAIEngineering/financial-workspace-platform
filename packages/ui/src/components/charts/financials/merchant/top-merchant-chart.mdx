# TopMerchantChart Component

The `TopMerchantChart` is a specialized visualization that focuses on displaying the top merchants by spending volume. This chart helps businesses quickly identify their most significant vendor relationships, providing insights for strategic procurement decisions, vendor consolidation opportunities, and potential areas for cost negotiation.

<Canvas of={TopMerchantChart.Default} />

```tsx

        { merchant: "Vendor E", amount: 53000 },

        { merchant: "Vendor J", amount: 35000 }

      currency="USD"
      locale="en-US"

      limit={5}
``` 
