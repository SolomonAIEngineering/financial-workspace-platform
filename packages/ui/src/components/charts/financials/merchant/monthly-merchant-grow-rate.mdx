# MonthlyMerchantGrowRate Component

The `MonthlyMerchantGrowRate` chart visualizes the month-over-month growth rates of spending with different merchants. This component helps businesses identify which vendor relationships are growing or shrinking over time, providing insights for vendor management and procurement strategies.

## Component Props

<Canvas of={MonthlyMerchantGrowRate.Default} />

<Controls />

## Usage Example

Here's a basic usage example of the `MonthlyMerchantGrowRate`:

```tsx
<Canvas>
  <Story name="Default">
    <MonthlyMerchantGrowRate 
      data={[
        { month: "Jan", vendorX: 0, vendorY: 0, vendorZ: 0 },
        { month: "Feb", vendorX: 4.5, vendorY: 2.8, vendorZ: 5.1 },
        { month: "Mar", vendorX: 3.2, vendorY: 3.6, vendorZ: 4.9 },
        { month: "Apr", vendorX: -1.5, vendorY: 2.1, vendorZ: 3.8 },
        { month: "May", vendorX: 2.8, vendorY: 1.5, vendorZ: 4.2 }
      ]}
      merchants={["vendorX", "vendorY", "vendorZ"]}
      displayNames={{
        vendorX: "Vendor X",
        vendorY: "Vendor Y",
        vendorZ: "Vendor Z"
      }}
      height={400}
      showAverageGrowth={true}
    />
  </Story>
</Canvas>
```
