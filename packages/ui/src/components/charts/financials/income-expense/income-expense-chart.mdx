# IncomeExpenseChart Component

The `IncomeExpenseChart` is a financial visualization that compares income and expenses side by side over time. This dual-metric chart helps users quickly understand their financial balance, identify periods of profit or loss, and analyze the relationship between earnings and spending patterns.

## Component Props

<Canvas of={IncomeExpenseChart.Default} />

<Controls />

## Usage Example

Here's a basic usage example of the `IncomeExpenseChart`:

```tsx
<Canvas>
  <Canvas of={IncomeExpenseChart.Default}>
    <IncomeExpenseChart 
      data={[
        { 
          date: new Date(2023, 0, 1), 
          income: 45000, 
          expenses: 38000 
        },
        { 
          date: new Date(2023, 1, 1), 
          income: 48000, 
          expenses: 40000 
        },
        { 
          date: new Date(2023, 2, 1), 
          income: 52000, 
          expenses: 39000 
        },
        { 
          date: new Date(2023, 3, 1), 
          income: 49000, 
          expenses: 41000 
        }
      ]}
      currency="USD"
      locale="en-US"
      height={400}
      showNetIncome={true}
    />
  </Canvas>
</Canvas>
```
