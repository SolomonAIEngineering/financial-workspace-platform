# TransactionsChart Component

The `TransactionsChart` visualizes transaction volume and patterns over time. This chart helps businesses understand their transaction flow, identify peak periods, and analyze trends in transaction frequency or volume. It can be used to optimize staffing, inventory, or operational resources based on transaction patterns.

## Usage Example

Here's a basic usage example of the `TransactionsChart`:

```tsx
<Canvas>
  <Story name="Default">
    <TransactionsChart 
      data={[
        { date: new Date(2023, 0, 1), count: 245, volume: 35000 },
        { date: new Date(2023, 1, 1), count: 278, volume: 42000 },
        { date: new Date(2023, 2, 1), count: 312, volume: 48500 },
        { date: new Date(2023, 3, 1), count: 290, volume: 44200 },
        { date: new Date(2023, 4, 1), count: 325, volume: 51000 }
      ]}
      currency="USD"
      locale="en-US"
      height={400}
      showCount={true}
      showVolume={true}
    />
  </Story>
</Canvas>
```
