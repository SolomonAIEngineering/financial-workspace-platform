# NetIncomeChart Component

The `NetIncomeChart` is a financial visualization that displays net income (profit) over time. This chart helps businesses track their bottom line, identify trends in profitability, and understand the relationship between revenue and expenses. It can include features like year-over-year comparisons, goal tracking, and forecasting.

## Usage Example

Here's a basic usage example of the `NetIncomeChart`:

```tsx
<Canvas>
  <Canvas of={NetIncomeChart.Default}>
    <NetIncomeChart 
      data={[
        { date: new Date(2023, 0, 1), netIncome: 12000 },
        { date: new Date(2023, 1, 1), netIncome: 15000 },
        { date: new Date(2023, 2, 1), netIncome: 13500 },
        { date: new Date(2023, 3, 1), netIncome: 16800 },
        { date: new Date(2023, 4, 1), netIncome: 18200 }
      ]}
      currency="USD"
      locale="en-US"
      height={400}
      showAverage={true}
      showYearlyComparison={false}
    />
  </Canvas>
</Canvas>
```
