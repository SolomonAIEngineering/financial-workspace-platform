# AnalyticsChart Component

The `AnalyticsChart` is a comprehensive chart component designed specifically for financial and business analytics. It supports multiple visualization types (bar, line, area), comparison features, and advanced filtering capabilities. The component includes automated insights, trend detection, and responsive design for enterprise-grade analytics dashboards.

## Usage Example

Here's a basic usage example of the `AnalyticsChart`:

```tsx
<Canvas>
  <Canvas of={AnalyticsChart.Default}>
    <AnalyticsChart 
      title="Revenue Analysis"
      subtitle="Monthly revenue breakdown"
      data={[
        { date: new Date(2023, 0, 1), revenue: 12000, expenses: 8000 },
        { date: new Date(2023, 1, 1), revenue: 14000, expenses: 8200 },
        { date: new Date(2023, 2, 1), revenue: 13500, expenses: 8400 },
        { date: new Date(2023, 3, 1), revenue: 15000, expenses: 8300 },
        { date: new Date(2023, 4, 1), revenue: 16500, expenses: 8600 }
      ]}
      chartType="bar"
      primaryMetric="revenue"
      secondaryMetric="expenses"
      currency="USD"
      showComparison={true}
      showInsights={true}
    />
  </Canvas>
</Canvas>
```
