# ZoomableChart Component

The `ZoomableChart` is an advanced chart component that allows users to zoom in and out of data for more detailed exploration. It's particularly useful for visualizing large datasets where details might be lost in a full view. The component includes pan and zoom controls, responsive behavior, and maintains context during navigation.

## Usage Example

Here's a basic usage example of the `ZoomableChart`:

```tsx
<Canvas>
  <Story name="Default">
    <ZoomableChart 
      data={[
        { date: new Date(2023, 0, 1), value: 1000 },
        { date: new Date(2023, 0, 15), value: 1200 },
        { date: new Date(2023, 1, 1), value: 900 },
        { date: new Date(2023, 1, 15), value: 1500 },
        { date: new Date(2023, 2, 1), value: 1300 },
        { date: new Date(2023, 2, 15), value: 1700 }
      ]}
      width={800}
      height={400}
      margin={{ top: 20, right: 20, bottom: 50, left: 50 }}
      xAxisDataKey="date"
      yAxisDataKey="value"
    />
  </Story>
</Canvas>
```
