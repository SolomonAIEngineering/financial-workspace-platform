# AreaChart Component

The `AreaChart` is a visualization component that displays data as a filled area beneath a line, making it ideal for showing data volume or cumulative values over time. It features customizable height, currency formatting, and responsive date filtering.

## Usage Example

Here's a basic usage example of the `AreaChart`:

```tsx
<Canvas>
  <Story name="Default">
    <AreaChart 
      currency="USD"
      data={[
        { date: new Date(2023, 0, 1), value: 1000 },
        { date: new Date(2023, 1, 1), value: 1500 },
        { date: new Date(2023, 2, 1), value: 1200 },
        { date: new Date(2023, 3, 1), value: 1800 }
      ]}
      height={300}
      locale="en-US"
    />
  </Story>
</Canvas>
```
