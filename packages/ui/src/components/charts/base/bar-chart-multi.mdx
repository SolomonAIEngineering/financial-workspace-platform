# BarChartMulti Component

The `BarChartMulti` is a specialized bar chart visualization that allows displaying multiple data series side by side. This makes it ideal for comparing different categories or metrics over the same time periods. The component features customizable colors, interactive tooltips, and responsive design.

## Usage Example

Here's a basic usage example of the `BarChartMulti`:

```tsx
<Canvas>
  <Story name="Default">
    <BarChartMulti 
      currency="USD"
      data={[
        { 
          date: new Date(2023, 0, 1), 
          income: 2000, 
          expenses: 1500 
        },
        { 
          date: new Date(2023, 1, 1), 
          income: 2200, 
          expenses: 1600 
        },
        { 
          date: new Date(2023, 2, 1), 
          income: 2400, 
          expenses: 1700 
        }
      ]}
      height={300}
      locale="en-US"
      keys={["income", "expenses"]}
    />
  </Story>
</Canvas>
```
