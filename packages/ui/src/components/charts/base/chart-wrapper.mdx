# ChartWrapper Component

The `ChartWrapper` is a higher-order component that provides common chart functionalities such as date range filtering, loading states, and assistant mode capabilities. It creates a consistent context for charts and exposes a state management hook called `useWrapperState` for child components to access shared chart state.

## Usage Example

Here's a basic usage example of the `ChartWrapper`:

```tsx
<Canvas>
  <Canvas of={ChartWrapper.Default}>
    <ChartWrapper 
      enableAssistantMode={true}
      initialDateRange={{
        from: new Date(2023, 0, 1),
        to: new Date()
      }}
    >
      {({ dateRange, isLoading }) => (
        <div className="p-4 border rounded">
          <p>From: {dateRange.from.toLocaleDateString()}</p>
          <p>To: {dateRange.to.toLocaleDateString()}</p>
          <p>Loading: {isLoading ? "Yes" : "No"}</p>
        </div>
      )}
    </ChartWrapper>
  </Canvas>
</Canvas>
```
