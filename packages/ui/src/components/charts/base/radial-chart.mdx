# RadialChart Component

The `RadialChart` is a circular visualization that displays data points arranged radially around a center point. This type of chart is particularly effective for showing cyclical data or comparative values across multiple categories. It features customizable colors, angles, and radius settings to create visually appealing data representations.

## Usage Example

Here's a basic usage example of the `RadialChart`:

```tsx
<Canvas>
  <Story name="Default">
    <RadialChart 
      data={[
        { subject: "Math", A: 120, B: 110, fullMark: 150 },
        { subject: "English", A: 98, B: 130, fullMark: 150 },
        { subject: "History", A: 86, B: 130, fullMark: 150 },
        { subject: "Physics", A: 99, B: 100, fullMark: 150 },
        { subject: "Geography", A: 85, B: 90, fullMark: 150 }
      ]}
      width={500}
      height={400}
      cx={250}
      cy={200}
      outerRadius={150}
    />
  </Story>
</Canvas>
```
